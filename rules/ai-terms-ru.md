# AI Терминология для русскоязычного контента

## Правила выбора языка

1. **Продукты** — всегда английский: Claude, GPT, Telegram, Python
2. **Акронимы** — английский: LLM, RAG, API, JSON, YAML
3. **Техники** — английский при первом упоминании: Zero-shot, Chain-of-Thought
4. **Общие концепции** — русский
5. **При сомнении** — русский
